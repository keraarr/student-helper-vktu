# Student Helper Bot

Telegram бот-помощник для студентов ЕКТУ, предоставляющий информацию о расписании и погоде.

## Функционал

- Выбор группы и сохранение расписания
- Просмотр расписания на сегодня, завтра, неделю
- Информация о погоде

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Настройте токен бота в `config/config.py`
4. Запустите бота: `python -m src.bot`

## Структура проекта

- `src/` - основной код
- `tests/` - тесты
- `config/` - конфигурация
- `scripts/` - вспомогательные скрипты
- `data/` - данные (база данных)
