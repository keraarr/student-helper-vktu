# Student Helper Bot

Telegram бот-помощник для студентов ЕКТУ, предоставляющий информацию о расписании и погоде.

## Функционал

- Выбор группы и сохранение расписания
- Просмотр расписания на сегодня, завтра, неделю
- Информация о погоде

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Настройте токен бота в `config/config.py`
4. Запустите бота: `python -m src.bot`

## Архитектура проекта

Проект построен по модульной архитектуре с разделением ответственности:

### Структура папок:
- `src/` - основной код приложения
  - `bot.py` - главный модуль, обработка команд Telegram
  - `schedule_parser.py` - парсинг расписания с сайта ЕКТУ
  - `db.py` - работа с базой данных SQLite
  - `weather_api.py` - интеграция с OpenWeatherMap API
- `config/` - конфигурационные файлы (токены, настройки)
- `tests/` - модульные тесты
- `scripts/` - вспомогательные скрипты для разработки
- `data/` - база данных и другие данные

### Диаграмма взаимодействия:

```
                           ПОЛЬЗОВАТЕЛЬ
                            (Telegram)
                                 │
                                 │ Сообщение
                                 ▼
                        ┌─────────────────┐
                        │   TELEGRAM BOT  │
                        │   (aiogram)     │
                        └────────┬────────┘
                                 │
                    ┌────────────┼────────────┐
                    │            │            │
                    ▼            ▼            ▼
            ┌──────────────┐ ┌─────────┐ ┌──────────────┐
            │ SCHEDULE     │ │  DB     │ │  WEATHER API │
            │ PARSER       │ │ (SQLite)│ │(OpenWeatherM)│
            │ (BeautifulSoup)
            └──────────────┘ └─────────┘ └──────────────┘
                    │            │            │
                    └────────────┼────────────┘
                                 │
                                 │ Ответ
                                 ▼
                           ПОЛЬЗОВАТЕЛЬ
```

### Поток данных:
```
Пользователь (Telegram)
    ↓
bot.py (обработка команд)
    ↓
schedule_parser.py (парсинг ЕКТУ) ↔ db.py (сохранение данных)
weather_api.py (погода)
    ↓
Ответ пользователю
```

### Технологии:
- **Python 3.10+** - основной язык
- **aiogram** - асинхронная библиотека для Telegram Bot API
- **BeautifulSoup + lxml** - парсинг HTML
- **SQLite** - база данных
- **requests** - HTTP запросы
